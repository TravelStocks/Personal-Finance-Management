# Personal Finance Management

一个长期个人财务管理网页，用来手动维护收入、支出、账户、资产分配、投资、应急金、提醒、目标和财务报表。

## 功能

- 总览 Dashboard：总资产、总储蓄、总负债、应急金、本月收入、本月支出、资产分配、6 个月预测。
- 收入管理：按月份记录工资、炒股月结、其他收入。
- 支出管理：按月份维护预算、实际支出、必要/可取消支出、固定支出。
- 现金流预测：未来 6 个月收入、支出、分配和预计余额。
- 账户管理：银行卡、支付宝、微信、证券账户、余额宝、现金等账户可新增、删除和编辑。
- 投资管理：A 股、美股、港股市值、成本、盈亏和汇率换算。
- 资产负债表、应急金、账单提醒、目标管理、财务报表、健康评分和未来数据能力模块。

## 本地运行

```bash
npm install
npm run dev
```

构建检查：

```bash
npm run build
```

## 隐私说明

页面默认使用脱敏账户名称，不包含原始截图、完整账号、微信号或账单文件。当前版本是静态网页和手动录入状态，刷新后不会持久保存运行时录入的数据。
